# Protocolos 
Protocolos são regras que definem o formato de transferência de informações.

## HTTP
_Hypertext Transefer Protocol_
Protocolo que define regras para transferência de hipertextos, que por sua vez são documentos que possuem referências de outros documentos.

## FTP
_File Transfer Protocol_
Protocolo dedicado para transferência de arquivos.

## SMTP
_Simple Mail Transfer Protocol_
Padrão utilizado para tranferência de mensagens de e-mail.
